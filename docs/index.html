常見問題

找到你需要的答案

語言包下載失敗怎麼辦？

請確認網路連線穩定，並重新嘗試下載。如果問題持續，請嘗試：

  • 檢查 Wi-Fi 連線是否正常
  • 重新啟動 App
  • 檢查裝置儲存空間是否足夠
  • 更新到最新版本的 App

離線翻譯不工作？

請確認已下載對應的語言包。前往「設定」>「語言管理」查看已下載的語言。每個翻譯方向都需要下載對應的語言包才能離線使用。

語音辨識不準確？

建議在安靜的環境中使用，並確保麥克風權限已開啟。說話時請保持正常語速，避免過快或過慢。

相機掃描無法辨識文字？

請確保光線充足，並將相機對準文字保持穩定。目前支援印刷體文字辨識，手寫文字可能無法正確辨識。

如何更改 App 介面語言？

App 會自動跟隨系統語言設定。如需更改，請前往 iPhone「設定」>「一般」>「語言與地區」進行調整。
